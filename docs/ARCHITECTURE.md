## System Overview
Bayesian-AI is a multi-timeframe trading system driven by a physics-inspired "Quantum Field" engine. Raw market data (from 1-second to Daily resolution) is processed into state vectors representing "Three-Body Quantum States" (fair value regression center and two repulsion singularities).

The system utilizes an 8-worker Timeframe Belief Network that aggregates signals across timeframes to compute "Path Conviction" (a weighted geometric mean of direction probability). Patterns are dynamically clustered via a Hypervolume Tree approach based on Individual-Moving Range (IMR) charts, enabling real-time template matching and execution.
