# 2026-04-05 — Bayesian Book Pipeline + Corrected Trades Fix

## Session Summary
Built the full Bayesian pipeline. Found and fixed the root cause of poor OOS performance.

## Key Finding
**Counter-flipping was destroying the edge.** Tree classified 43% of trades as "counter" and AI flipped their direction. Counter-flipped trades only won 54% (coin flip). Same-direction trades won 69%. The fix: regret produces CORRECTED trades (right direction, right exit, real PnL from prices), tree trains on those. No more counter-flipping.

## Results
| | Before Fix | After Fix (H0) |
|---|---|---|
| IS $/day | $5 | **$62** |
| IS Win % | 38% | **63%** |
| OOS $/day | -$16 | **$51** |
| OOS Win % | negative | **52%** |

H1 (after epoch learning): $55/day IS, $44/day OOS — epochs slightly hurt. H0 baseline is the winner. Epoch learning needs refinement.
