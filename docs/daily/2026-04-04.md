## Architecture Decisions
- Ticker is dumb pipe (1s bars only)
- Aggregator handles all TF construction + events
- SFE is the source of truth for 79D (live or test mode)
- NMP only reads 79D — never computes features
- No black boxes: tree splits are readable if/then rules
- Tree = base strategy selector, NN = refinement (not built yet)
- Segment regression = trade health monitor (not built yet)

## Key Rule Established
- Build dataset: bulk GPU, no lookahead protection needed (it's training data)
- Live/test ticker: sequential 1s bars, zero lookahead by construction
- Every module only reads data from the layer above it
