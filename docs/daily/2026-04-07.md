## Key Discovery: CNN > Tree for Multi-Feature Patterns
The tree splits on ONE feature at a time. The CNN convolves across the full 6×13 grid.
"z extreme + wick + 1h aligned" is ONE spatial pattern to the CNN but THREE independent
splits for the tree. This is why CNN gets 70.6% where tree gets 55.5%.

## Stage 2 Architecture (next session)
Not a TRADE/SKIP classifier. A physics discovery engine:

1. Run blended+CNN → trades
2. Regret on those trades → corrected trades (optimal entry/exit)
3. Extract 79D at optimal entry points (earlier entries, different physics)
4. CNN Stage 2 learns: what 79D patterns at these optimal entries predict profit?
5. Cluster new entries → find unique physics that generalize
6. Test each cluster on full IS → only generalizing physics survive
7. Surviving clusters → new ExNMP strategies → expand blended engine
8. Repeat

This is the real epoch learning — discovering new physics, not tweaking exit bars.
