# 2026-05-26: V2 Native Architecture Migration & 8-Agent DOE Sweep

## 1. What Changed
- **OOS Walk-Forward Validation**: Integrated a rigid out-of-sample forward pass into `train_historical.py`. The model must now prove generalization on an unseen subsequent week of data before it is allowed to graduate.
- **Dynamic Pity Pass**: Implemented a curriculum learning adaptation where if a model fails to meet the static OOS metric after 30 epochs, the threshold dynamically adjusts to `average_oos_last_30_epochs + 10%`. This prevents infinite loops on mathematically impossible market regimes while still requiring an upward trajectory.
- **8-Agent DOE Orchestrator**: Refactored the training engine to support dynamic agent architectures (`--agent-type`). Spun up a multi-process orchestrator (`run_doe.py`) to launch 8 concurrent trials across two learning rates (3e-3, 4e-3) testing four distinct architectures: `NMP`, `ENTRY_NMP`, `EXIT_NMP`, and `YOLO`.
- **HFT Guardrails**: Hardcoded severe asymmetric reward shaping for all models (including `YOLO`) to penalize any trade with a PnL < $20, enforcing a strict ban on high-frequency noise scalping to protect limited account equity.

## 2. Files Modified/Created
- `training/rl_engine/train_historical.py` [MODIFIED]: Added dynamic architecture support, OOS validation logic, and memory cleanup (`gc.collect()`, `torch.cuda.empty_cache()`).
- `training/rl_engine/run_doe.py` [NEW]: Multi-process Python orchestrator script to manage the concurrent sweep.

## 3. What to Look For in the Next Run
- **HDF5 Serialization Integrity**: Monitor the `experiences_*.h5` shadow queues generated by the 8 concurrent agents to ensure they successfully flush terminal states to disk without File I/O collisions.
- **DOE Survival**: Ensure the "smoke test of the smoke test" completes its 3-epoch boundary across all 8 agents successfully without memory leaks or CUDA context crashes.
- **Agent Convergence Variance**: We expect to see drastically different PnL profiles between the `ENTRY_NMP` (constrained by structural exits) and the `YOLO` agents (pure unconstrained neural gymnastic exploration).
